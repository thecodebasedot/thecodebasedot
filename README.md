## ◆ About

> **The Codebase** is an engineering portfolio focused on intelligent systems, scalable data
> platforms, and security-first software infrastructure. Every repository is built to reflect
> **research-driven engineering**, **production-oriented architecture**, and **modular system design**.

- 🧠 Building **AI / ML systems** — from data ingestion to model integration
- 🛡️ Designing **secure-by-default** software and monitoring frameworks
- 🏗️ Engineering **distributed data pipelines** and analytical infrastructure
- 📍 United Kingdom · Open to research & engineering collaboration
